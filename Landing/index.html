O kursie

Ten kurs jest kompleksowym wprowadzeniem do języka programowania Java – jednego z najpopularniejszych i najbardziej uniwersalnych języków na świecie. Program szkolenia obejmuje zarówno solidne fundamenty (składnia, typy danych, struktury kontrolne), jak i zaawansowane zagadnienia, takie jak programowanie obiektowe, obsługa wyjątków, kolekcje, programowanie wielowątkowe oraz podstawy Spring Framework. Zajęcia prowadzone są w formie interaktywnych wykładów i ćwiczeń praktycznych, co pozwala uczestnikom natychmiast zastosować zdobytą wiedzę. Po ukończeniu kursu każdy uczestnik będzie w stanie tworzyć własne aplikacje w Javie oraz zrozumie architekturę projektów komercyjnych.

Program

  • Podstawy Javy i środowiska (IntelliJ IDEA)
  • Programowanie obiektowe: klasy, dziedziczenie, interfejsy
  • Baza danych: JDBC, podstawy SQL
  • Wprowadzenie do Spring Boot i REST
